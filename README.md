# yahoo-weather

仿雅虎天气 iOS 客户端

### Quick Start

* Clone this repo
* Edit `Podfile` and remove the `Reveal` related plugin if you want to save some time
* Run `Pod install`
* Get a cup of coffee or power on your Xbox and wait for its stupid updating
