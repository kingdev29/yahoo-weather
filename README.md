# yahoo-weather

仿雅虎天气 iOS 客户端

### Quick Start

* Clone this repo
* Run `Pod install`
* Get a cup of coffee or power on your Xbox and wait for its stupid updating
* Add your [OpenWeatherMap API](http://openweathermap.org) to `YWClient.m`
* Build and Run
